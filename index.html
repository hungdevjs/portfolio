Dance school

Project goal

BASE32.se is a famous dance school in Stockholm, Sweden, and belongs to the government.
They have used an access database file to manage students, classes for over 20 years.
My team's mission is to build a brand new school management system for them that they can access from everywhere.

Project description

We have built from the ground up a new school management system on web and mobile app using latest technologies such as on Azure cloud.
Our system can provide all common functionalities of a school management system such management of students, teachers, classes, courses, families.

Our web app also provide features such as course scheduling, absent reports, payment tracking, mass email module, integration to invoice and salary system Fortnox, importing students via WordPress webhooks.

Technology stack

Database: Microsoft SQL Server
Back-end Server: C# .NET Core Entity Framework
Web Front-end: React.js
Mobile Application: React Native
